## 前言

臺科的選課環境對學生相當友善，比起其他學校來說，我們擁有更多的資源和紀錄，像是**歷年成績**、**課程評價**、**考古題**...，只要願意花一點時間做功課，就能排出讓自己滿意的課表喔！ ouob

:::tip
如果不知道自己在幹嘛，那至少記住三件事
1. 志願序填體育和國文
2. 學校預排好的課不要亂退
3. 志願序多填不吃虧
:::

## 選課時程

:::steps
1. **新生及轉學生初選** | **上機登記**    
    將想修的課排**志願序**，時間截止後交給電腦抽籤
2.   ＿＿＿＿＿＿＿＿   | **查詢結果**  
    登入選課網站，查詢抽籤結果
3.   ＿＿＿＿＿＿＿＿   | **繼續選課**  
    所有人一起即時加選或退選[^1]，熱門的課考驗先搶先贏
4. **全校加退選**  
    所有人一起即時加選或退選，熱門的課考驗先搶先贏。**臺大與師大的課程開放選課**
5. **加簽**  
    當課程無法選、人數滿時，直接向教授要**授權碼**加簽[^2]
6. **二次退選**  
    對期中考成績不滿意或快被當時，將課程退掉的方式[^3]
:::

[^1]: 也可以和朋友交換課！
[^2]: 通常會先寄信知會教授，或直接去旁聽課程，下課後找教授；電腦教室等受限的空間，通常不允許加簽！
[^3]: 二退與被當掉相比，被當掉的嚴重性大很多喔！

## 課程資訊

### 基本名詞

| 名詞 | 意思                                           |
|---|----------------------------------------------|
| 學分 | 課程份量的單位，約等於每週上課時數；畢業門檻就是「修滿規定組合的學分」          |
| 必修 | 畢業前一定要修過的課。全校共同的（國文、英文、通識、體育）叫共同必修，系上規定的叫系必修 |
| 選修 | 自由選擇的課；常見的有系選修、院選修。畢業前要修滿規定學分                |
| 志願序 | 初選時你排出的期望清單，抽籤依這個順序進行，越前面越優先                 |
| 衝堂 | 兩門課上課時間重疊。在志願抽選時，僅會抽中一堂，其餘將會自動放棄             |
| 擋修 | 課程設有先修條件或身分限制[^4]，不符合系統不給選。可以透過授權碼**加簽**來繞過  |

[^4]: 若是研究所課程，[系統規定](https://www.academic.ntust.edu.tw/var/file/48/1048/img/2570/830893402.pdf)至少要大三才能夠修習，加簽也沒辦法喔！

### 課程代碼

課程代碼是一串英數字，以「CT1003301」為例

| 位置 |  範例  | 意思 |
|---|:----:|---|
| 第 1～2 碼 |  CT  | 開課系所（CT 營建系）；3T 開頭是臺大課程、3N 是師大課程 |
| 第 3 碼 |  1   | 學制：1～4 是大學部、5～9 是研究所、G 是大學部通識 |
| 第 4～6 碼 | 003  | 系所自訂流水號 |
| 第 7 碼 |  3   | 必選修：0 全學年必修、3 半學年必修、5 全學年選修、7 半學年選修 |
| 第 8～9 碼 |  01  | 開課班次 |

> 通識課的代碼是「GE」開頭（人文社會學科開設），或第 3 碼為「G」（其他教學單位開設）

### 想清楚要什麼

同一門課對不同人可能是天堂也可能是地獄

- 修自己有興趣的課
- 可以睡到飽
- 不容易被當
- 容易拿高分
- 沒有作業、沒有考試
- 不用上臺報告、不想分組

## 學分限制

- 大一到大三每學期可選 16～25 學分（大四下限是 9 學分）
- 若前一學期 GPA 達 3.38，最多可超修至 31 學分，但只能在全校加退選階段辦理
- 志願序的總學分**可以**超過 25，抽選中滿 25 學分會自動停止

:::warning
下限 16 學分非強制，沒修滿的後果是該學期不記名次，名次會影響之後推甄研究所
:::

## 各類課程怎麼選

### 系必修：已經幫你排好

- 各班必修皆預先載入，不用選就在課表上
- 所有志願都要避開這些時段

:::warning
不建議為了搶其他課退掉系必修。如果又再多二退了，一不小心就會延畢 ><
:::

### 體育：固定班底

- 體育 0 學分，但**大一到大三每學期都必修**，每學期限修一門，共六次
- [大一體育](https://dss20.ntust.edu.tw/edua/courlist/qry_coursebook.aspx?fid=2&sid=1&tid=1)多在第 9、10 節（16:30～18:20），星期幾由各系分配，只能選自己系/班級被分配到的項目；體育老師通常不接受跨時段加簽
- 在課程查詢系統的「體育課程」分頁可以查**體育課程時間表**與**必修課程人數分配表**
- 體育是全學年課程，下學期將系統直接帶入，不用重選；也**不會與系必修衝堂**
- 初選最多中一門，最少零門，所以**多填幾門當備胎**
- 沒抽中的話，可以在加退選找沒有滿人加選

### 國文：大一修完

- 共兩堂順序自選，「新生中文說寫專題」與「表達與文學閱讀 / 表達與經典閱讀 / 創意與文學閱讀 / 創意與經典閱讀 擇一」
- 國文依各系排定的時段開課，從班級課表看有哪些時段可挑
- 建議大一就把兩門修完
- 初選最多中一門，最少零門，所以**多填幾門當備胎**
- 沒抽中的話，可以在加退選找沒有滿人加選

### 英文：需要運氣

- 英文門檻 12 學分，入學時依英文程度分入基礎、中階、進階模組，之後照模組規定修課
- 另有 0 學分的「校定英文能力檢測」畢業門檻，未通過需加修「英文實務」
- 分級方式二選一
  - 參加**入學英文分級會考**
  - 持官方英檢成績（CEFR 聽讀 B1 以上，例如多益 550、全民英檢中級）申請免考
- 大一共同必修英文（英文字彙與閱讀、英語口語訓練）**已預設在課表上**，課碼是暫定的，分班名單開學前公告
- 「多元英文必修」（自選英文）要等加退選才能加選；熱門課輪不到新生，大一下、大二再修完全來得及

:::tip
確定英文學分能抵免的人，可以在初選就退掉預設英文，把時段讓給其他課。但**務必百分之百確認抵免會過**再退，不確定就不要動。
:::

:::danger
入學前沒申請免考、也沒參加分級會考的話，會因為未分級而修不了任何英文課，最快要等升大二暑假的會考後才能開始修
:::

### 通識：是涼還是水

- 通識門檻 15 學分，六個向度至少跨四個：A 人文素養、B 當代文明、C 美感與人生探索、D 社會與歷史文化、E 群己與制度發展、F 自然與生命科學
- ==初選階段通識最多抽中三門==（含繼續選課），全校加退選才解除限制
- 部分通識在初選設有**優先選課班級**：排進你們班課表通識時段的課，抽選時你們班優先，除非登記人數超過保留名額才需要抽。先查班級課表，這些課中籤率最高
- 判讀名額：還沒開放選課時，「人數上限－選課人數」是剩餘名額；初選期間顯示的則是「把它加入志願序的人數」，只能當熱門度參考
- 好通識大多被舊生選走了，等大一下、大二撿好課完全來得及；通識成績通常是拉高 GPA 的主力，值得花時間挑
- 對臺大、師大的課有興趣，要等全校加退選；向度與學分認列方式選課前先向教務處確認

:::dept{for="csie"}
- **系選修**：畢業需要 17 學分，但本屆大一課表上沒有開給新生的系選修；想提前偷跑，開學後可以找教授加簽大二、大三的課
- **專業必修三選一**：「電資概論與工程倫理」「臺灣電資產業論壇與工程倫理」「電資工程倫理與法律實務」三門擇一修過即可，新生初選通常抽不中，之後學期再修就好
- 先看過 [資工系課程大綱](https://www.csie.ntust.edu.tw/p/412-1038-2146.php?Lang=zh-tw) 與 [資工系課程地圖](https://www.academic.ntust.edu.tw/var/file/48/1048/img/3313/660287396.jpg)，知道四年會遇到哪些課，規劃學分才能贏在起跑點
:::

## 志願序怎麼填

上機登記期間唯一要做的事：把想修的課依偏好排成志願序，等電腦抽籤。規則重點：

- 志願上限 **30 個**，總學分可以超過 25，如果抽到不想要的之後再退
- 某志願中籤後，清單裡**同課名**與**同時段**的其他志願自動作廢，所以不會重複中籤、不會衝堂
- 熱門課不放前段幾乎不會中；要嘛放前面，要嘛乾脆不放
- 體育與國文**千萬不要只填一門**
- 搭配[志願序助手](https://github.com/NTUST-Tool/Course-order-assistant/)可以算出期望值最高的排法

## 加退選：魔法師

全校加退選從開學前一週左右跑到開學第二週結束，與臺大系統課程同時開放，即時加退、先搶先贏

- 好課基本上一路額滿，要撿漏就得**蹲點**：守在課程查詢系統前反覆重新整理，有人退的瞬間馬上加
- **上課當天**最容易有波動（有人試聽完決定退課），開學前幾天與加退選截止前也是高峰
- 退出的名額全校都能搶，因此跟同學約好「你退我加」交換課程的風險很高
- 通識此時不限系級全面開放，初選的三門上限也解除
- 臺大、師大課程新生從這個階段起才能選；三校以外的學校要填「校際選課單」人工辦理
- 開學後可以先去試聽再決定退不退，等於每門課都有鑑賞期
- 加退選截止後的「選課更正期間」只處理選課錯誤，不能藉機加退選，所以截止前務必自己確認課表

## 加簽與授權碼：教授救我

課滿了、被擋修，或錯過所有階段，最後手段就是直接向授課教師請求**加簽**

- 加簽潮在開學前兩週：通常要親自去該課的第一堂課向老師詢問，也可以先寄信問加簽規則（有些老師大四優先、要先交心得、限額發放）
- 不符合選課審核規則（衝堂、超過學分上限等）的加簽無效，就算輸入了授權碼，教務處也會逕行退選

:::danger
領授權碼時老師會要你**簽名**，這組授權碼只屬於簽名的人，**不能轉讓**。被發現轉讓，老師有權直接把你退選。
:::

## 二次退選：後悔藥

期中考後還有一次退場機會，俗稱**二退**

- 用途是停損：期中砸了、確定救不回來，退掉能避免成績單出現不及格
- 流程是到學生資訊系統申請並列印申請單，送註冊組（行政大樓 1 樓）取得收據才算完成
- 退選後學分不得低於下限（大一 16 學分），退太多系統會擋
- 已繳的學分費不退；短期密集課程、彈性學分課程不能二退

:::warning
二退的科目會在**學期成績單註記「退選」**。比起不及格當然是小傷，但常態性二退在推甄審查時不好看；而且退掉的學分之後都要補修，補太多學期塞不下就會**延畢**
:::

## 常見問題

:::qa[志願序最多能填幾個？]
30 個，總學分可以超過 25，新生抽滿 25 學分會自動停止
:::

:::qa[會不會同一時段中兩門課，或同一門課中兩次？]
不會。任一志願中籤後，同課名或同時段的其他志願都會自動作廢
:::

:::qa[志願序要按送出嗎？]
不用，課程加入志願序就代表登記成功，沒有另外的送出按鈕
:::

:::qa[想選的課顯示「已達人數上限」怎麼辦？]
那是舊生初選就選滿的課。等全校加退選蹲有沒有人退，或開學找老師加簽
:::

:::qa[新生選不到英文課正常嗎？]
正常。大一必修英文由學校預設與分班，不用自己選；自選英文要等加退選之後
:::

:::qa[體育可以選其他時段嗎？]
不行，只能選自己班級被分配到的時段與項目；跨時段需要老師加簽，而體育老師通常不給
:::
