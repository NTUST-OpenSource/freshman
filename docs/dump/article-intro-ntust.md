# 認識臺科

## 關於臺科
🔖 [簡介  - 國立臺灣科技大學](https://www.ntust.edu.tw/p/412-1000-95.php?Lang=zh-tw)
🔖 [國立臺灣科技大學 - 維基百科，自由的百科全書](https://zh.wikipedia.org/zh-tw/%E5%9C%8B%E7%AB%8B%E8%87%BA%E7%81%A3%E7%A7%91%E6%8A%80%E5%A4%A7%E5%AD%B8) — 國立臺灣科技大學（簡稱臺灣科大、臺科大）是臺灣第一所技術實務型高等教育機構[11]，1974年立校，也是唯一同時列入邁向頂尖大學計畫與發展典範科技大學計畫的學府[12]，主校區位於臺北公館，除了與臺灣大學及臺灣師範大學密切合作、合組可互相雙主修及輔系的臺大系統外，另設有東京工業大學的臺灣交流中心[13]。
